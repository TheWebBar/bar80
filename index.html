My Work

Project 01: Barryshop

This is a project I realised in Ruby on Rails using Heroku Postgres as a database for the backend part.

Barryshop is a web app offering outdoor equipment. The App is designed to allow the user to quickly and intuitively find the things he needs.

The objective was to create a web app with a database connection that allows the user to sign up and in, to write and edit/delete product reviews, to send mails via the web app and to order stuff. It furthermore allows the admin to delete comments and to add and edit products.

The goal was to create a simple app with an index-, contact-, products- and order-page that is easy to extend and that later allows the UX/UI-Designer to implement their styling demands.
